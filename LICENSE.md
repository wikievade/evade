- Wait 5–10 seconds; the GUI loads after the game fully initializes
- Ensure your executor supports the `Drawing` and `CoreGui` libraries

**Auto bhop feels inconsistent or stutters**
- This usually happens when frame rate drops below 30 FPS; lower Roblox graphics settings
- Make sure no other bhop-related script is loaded at the same time

**Auto revive not triggering**
- Confirm the toggle is ON in the GUI
- Rejoin the server — some map variants require a fresh injection

**Infinite slide cancels after one use**
- This is caused by a game update changing slide cooldown values; update to the latest
  script version

**Auto collect candy misses some drops**
- Increase your executor's script execution priority if the option is available
- Ensure you're not using a heavily modded executor build with stripped libraries

**Teleport sends player out of bounds**
- Coordinate tables update with every map rotation; always use the latest script version
